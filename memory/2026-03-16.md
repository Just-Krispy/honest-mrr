## BREAKTHROUGH IDEA: Cassandra Brain Integration

### The Vision (March 16, 03:39-03:43 UTC)
**Core Insight:** Integrate Cassandra AI (game theory predictions) with Second Brain (knowledge graph) to create a **Predictive Intelligence Platform**.

### Key Synergies Identified:

#### 1. Historical Pattern Matching
- Second Brain stores historical scenarios
- Cassandra AI uses patterns to refine predictions
- Example: "China-Taiwan conflict" → finds Cuban Missile Crisis, Ukraine parallels → improves probability estimates

#### 2. Entity-Driven Scenario Generation
- Second Brain tracks entities (people, companies, technologies)
- New connections auto-generate Cassandra scenarios
- Example: "Tesla + Taiwan Semiconductors + China Tensions" → auto-scenario

#### 3. Decision Tracking & Backtesting
- Cassandra makes predictions with timestamps
- Second Brain stores predictions in vault
- 6 months later: check accuracy (Brier score)
- Learn which scenario types work best

#### 4. Knowledge-Enhanced Prompts
- User asks Cassandra a question
- Second Brain injects relevant vault context
- Richer information = better analysis

#### 5. Temporal Reasoning Engine
- Second Brain shows timestamped notes (trends over time)
- Cassandra models how situations evolve
- Detect inflection points, cascading effects

#### 6. Interactive Decision Trees with Vault References
- Cassandra generates visual decision trees
- Each node links to Second Brain notes
- Navigate tree while reading supporting research

#### 7. Multi-Scenario Simulation Runner
- Generate 10+ scenarios automatically
- Run through Cassandra in batch
- Second Brain visualizes outcomes in graph
